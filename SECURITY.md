# Security Policy

## Supported versions

Temporaless is pre-1.0. Security fixes are provided only for the latest tagged
release line and the current `main` branch. Older release lines are unsupported.

## Reporting a vulnerability

Please email [jjkoh95@gmail.com](mailto:jjkoh95@gmail.com) with the subject
`Temporaless security report` instead of opening a public issue. Include the
affected tag or commit SHA, a minimal reproduction, the expected impact, and
any known mitigations. Do not include credentials or production data that are
not necessary to reproduce the issue.

Temporaless packages are distributed from Git only. Reports should identify the
exact immutable commit used by the affected deployment.
